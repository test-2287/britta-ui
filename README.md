# britta-ui component demo lib

### start

#### 1.install
``` bash
npm i britta-ui
```

#### 2. use
```javascript
// import all
// import css
import 'britta-ui/dist/css/index.css';
import BUI from 'britt-ui';
Vue.use(BUI)

// import by use
import 'britta-ui/dist/css/demo.css';
import { Demo } from 'britta-ui';
Vue.use(Demo)
```
