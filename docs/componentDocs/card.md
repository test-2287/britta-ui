# Card 
卡片组件

### 实例
<m-card imgSrc="/britta-ui/a.png" summary="test"/>

### 代码
```html
<m-card imgSrc="/britta-ui/a.png" summary="test"/>
```

### Attributes
| 参数 | 说明 | 类型 | 必要 | 默认值 |
| --- | --- | --- | --- | --- |
| width | 卡片宽度 | Number | false | - |
| imgSrc | 图片资源 | string | true | - |
| imgHeight | 图片高度 | Number | false | - |
| summary | 卡片概要 | String/slot | false | - |
| footer | 卡片地步 | slot | false | - |
